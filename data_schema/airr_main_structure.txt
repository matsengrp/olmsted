AIRR Main Data Structure
========================
Based on: full_schema_dataset.json

Root Object
├── ident: string (dataset unique identifier)
├── build: object
│   ├── commit: string (git commit hash)
│   ├── id: string (build identifier)
│   └── time: string (ISO timestamp)
├── subjects: array
│   └── [subject]: object
│       ├── subject_id: string
│       └── ident: string (unique identifier)
├── samples: array
│   └── [sample]: object
│       ├── locus: string (e.g., "igh")
│       ├── ident: string (unique identifier)
│       ├── timepoint_id: string
│       ├── sample_id: string
│       └── partitions: array
│           └── [partition]: object
│               ├── step: integer
│               ├── ident: string (unique identifier)
│               ├── logprob: number (log probability)
│               ├── id: string (partition identifier)
│               └── partition-file: array of strings
├── seeds: array
│   └── [seed]: object
│       ├── ident: string (unique identifier)
│       └── seed_id: string
└── clones: array
    └── [clone]: object
        ├── clone_id: string (primary identifier)
        ├── sample_id: string (foreign key to sample)
        ├── dataset_id: string (foreign key to dataset)
        ├── subject_id: string (foreign key to subject)
        ├── seed_id: string|null (foreign key to seed)
        ├── type: string (e.g., "cft.cluster")
        ├── size: integer (clone size)
        ├── rearrangement_count: integer
        ├── unique_seqs_count: integer
        ├── total_read_count: integer
        ├── v_call: string (V gene assignment)
        ├── d_call: string (D gene assignment)
        ├── j_call: string (J gene assignment)
        ├── v_alignment_start: integer
        ├── v_alignment_end: integer
        ├── d_alignment_start: integer
        ├── d_alignment_end: integer
        ├── j_alignment_start: integer
        ├── j_alignment_end: integer
        ├── junction_length: integer
        ├── junction_start: integer
        ├── germline_alignment: string (full sequence)
        ├── germline_alignment_length: integer
        ├── mean_mut_freq: number (mutation frequency)
        ├── path_length: number
        ├── has_seed: boolean
        ├── tree_count: integer
        ├── v_per_gene_support: array
        │   └── [support]: object
        │       ├── ident: string (unique identifier)
        │       ├── gene: string (gene name)
        │       └── prob: number (probability)
        ├── j_per_gene_support: array
        │   └── [support]: object
        │       ├── ident: string (unique identifier)
        │       ├── gene: string (gene name)
        │       └── prob: number (probability)
        └── trees: array
            └── [tree]: object
                ├── ident: string (tree unique identifier)
                ├── tree_id: string (tree identifier)
                ├── newick: string (newick format tree)
                ├── downsampling_strategy: string
                └── nodes: object (map of node_id -> node_data)
                    └── [node_id]: object
                        ├── sequence_id: string
                        ├── parent: string|null (parent node ID)
                        ├── type: string ("leaf" | "internal")
                        ├── distance: number (distance from root)
                        ├── length: number (branch length)
                        ├── multiplicity: integer
                        ├── cluster_multiplicity: integer
                        ├── sequence_alignment: string
                        ├── sequence_alignment_aa: string
                        ├── timepoint_id: string
                        ├── lbi: number (local branching index)
                        ├── lbr: number (local branching ratio)
                        ├── affinity: number|null
                        ├── timepoint_multiplicities: array
                        │   └── [timepoint]: object
                        │       ├── timepoint_id: string
                        │       └── multiplicity: integer
                        └── cluster_timepoint_multiplicities: array
                            └── [timepoint]: object
                                ├── timepoint_id: string
                                └── multiplicity: integer
